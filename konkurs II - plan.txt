konkurs II


prezentować bez włączonej grafiki w tle

aplikacja:
1. obliczanie punktów na krzywej Beziera		Python - numpy
2. rysowanie						Python - matplotlib
3. interfejs						Python - customtkinter



1. alg do obliczania punktów na krzywej Beziera  
- "Linear-time geometric algorithm for evaluating Bezier curves" - Paweł Woźny, Filip Chudy

2. Opcje rysowania:
- Dodaj / Usuń punkt / Przesuń punkt
- Dodaj / Usuń / Przesuń krzywą
- Dodawanie i usuwanie klatki
- Zmiana grubości krzywej
- Zmiana koloru danej krzywej i tła każdej klatki (jednocześnie, planowane było przypisanie tła do klatki)

3. Opcje interfejsu
- Nowy projekt
- Zapisz (i przy wychodzeniu)
- Eksportuj klatki
- Eksportuj jako gif
- Undo, Redo (Nie zaimplementowane)
- Wgrywanie grafiki w tło + checkbox 'pokaż grafikę' (gdy grafika włączona o wiele bardziej laguje)
- Wybór krzywej do edycji
- Wybór klatki do edycji
(+) Poprzednia klatka widoczna minimalnie w tle + checkbox 'poprzednia klatka' (gdy grafika włączona o wiele bardziej laguje)
